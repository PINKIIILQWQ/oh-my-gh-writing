# 案例库

这个目录用于校准 `oh-my-gh-writing` 的输出风格。每个场景文件收集真实 GitHub 仓库的写法，包含来源链接、摘录和结构分析。

## 查看方式

- 在 GitHub 中打开本目录或任一 `.md` 文件，可以直接查看案例页的 Markdown 渲染效果。
- README 场景额外提供“渲染效果”链接，跳到原仓库 README 的最终展示页；“完整内容”链接保留 raw Markdown，便于比对源文本。
- `reference/` 仍是正式写作规则；本目录是案例辅助材料，不替代场景标准。
