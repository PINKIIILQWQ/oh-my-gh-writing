## Applicability

This project is intended for AI agents and rule-based coding tools that can read a local `SKILL.md` file plus supporting Markdown references.

| Use mode | Best for | Limit |
| --- | --- | --- |
| Skill directory | Hosts that load local Agent Skills | Requires local file access |
| Project rules | Tools with custom instruction files | Needs manual adaptation |
| Knowledge base | Teams that search Markdown docs | Routing depends on the host |

Unsupported or unverified hosts should be marked `To confirm` instead of listed as supported.
