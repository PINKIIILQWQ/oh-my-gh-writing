# 测试输入：PR Template

## 用户请求

```
请为一个真实 Rust CLI 仓库创建 PULL_REQUEST_TEMPLATE.md。

要求：
1. 先读取目标仓库的 README、Cargo.toml、CI workflow、测试命令和 `.github/` 目录。
2. 再读取 1-2 个同类 Rust CLI 仓库的 PR 模板。
3. 模板中的测试 checklist 只能使用目标仓库能证实的命令。
4. 不要预先勾选 checklist，不要填入虚构 issue 编号或 benchmark。
```

使用 oh-my-gh-writing 的 PR Template 场景。
